# Документація маршрутів та API сайту Shadow Fight 2

## 1. Структура сайту

Сайт складається з двох основних сторінок:

-   **index.html** --- головна сторінка
-   **second_page.html** --- друга сторінка

------------------------------------------------------------------------

## 2. Маршрути (Routes)

  Route / URL             Тип      Опис
  ----------------------- -------- -------------------------------
  `/` або `/index.html`   GET      Головна сторінка сайту
  `/second_page.html`     GET      Перехід на другу сторінку
  `#home`                 Anchor   Головний банер, трейлер
  `#about`                Anchor   Опис гри
  `#services`             Anchor   Галерея скріншотів
  `#contacts`             Anchor   Контакти, магазини, соцмережі

------------------------------------------------------------------------
